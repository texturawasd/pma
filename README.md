# pma
programa para abstraer la gestión de paquetes en cualquier sistema (capa de abstracción)

## Version 0.2.0:
Por ahora soporta Linux y FreeBSD (y teoricamente otros sistemas, pero no se ha testeado), y determina que gestor de paquetes usar chequeando el PATH.
(nota: solo dispongo de sistemas: arch linux, ubuntu, FreeBSD; para testear)

### notas:
se cambia de "sys_update" a "pma" (Package Manager Abstraction)
